A. Bottom Tab Bar

Thanh tab co dinh o duoi man hinh. Chuyen trang nhanh bang 1 tap. Pho bien nhat cho mobile app (Instagram, Grab...)

Pros

  • Quen thuoc voi user
  • Chuyen tab nhanh 1 cham
  • Luon thay cac muc chinh

Cons

  • Gioi han 4-5 tab
  • Manager can nhieu muc hon

B. Bottom Tab + FAB (Check-in)

Bottom tab cho navigation + Floating Action Button noi bat o giua cho Check-in/Check-out. Hanh dong chinh luon noi bat.

Pros

  • Check-in noi bat, de thao tac
  • Van co tab navigation
  • UX tot cho hanh dong hang ngay

Cons

  • FAB chiem khong gian
  • Phuc tap hon option A

C. Hamburger Menu + Dashboard Grid

Menu hamburger (slide-out) + Dashboard dang luoi. Linh hoat, hien thi duoc nhieu muc hon. Phu hop khi Manager co nhieu chuc nang.

Pros

  • Khong gioi han so luong menu
  • Dashboard tong quan
  • Manager thay ngay "Duyet phep"

Cons

  • Hamburger can 2 tap de chuyen trang
  • Menu bi an, user co the quen
